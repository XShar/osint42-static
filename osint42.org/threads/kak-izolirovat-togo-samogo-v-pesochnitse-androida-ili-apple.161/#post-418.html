Заметка Как изолировать того-самого в песочнице Андроида или Apple

  • Автор темы 0x42
  • Дата начала 26.08.2025
  • 26.08.2025
  • Ответы: 4


Вот вы счастливый работник на заводе, учитель или ох-ёпрст депутат...)

Вам настоятельно попросили установить того-самого, кого не называют и отказаться нельзя...

Что делать, покупать отдельный телефон ?
Хороший вариант, но есть менее радикальный, предлагаю установить песочницу на телефон, также такую песочницу можно использовать и для других программ...)

Зачем нужна изоляция мессенджера?​

Мессенджеры, включая Max, получают широкий доступ к данным устройства:
  • Контакты
  • Хранилище
  • Геолокация
  • Камера и микрофон
  • Идентификаторы устройства
Изоляция приложения в песочнице:
  • Защищает личные файлы и контакты.
  • Снижает риск утечек данных.
  • Позволяет иметь чистое окружение без кэшей и следов.

Изоляция Max на Android​

На Android есть несколько способов сделать песочницу: системная, виртуальная и контейнерная.

Разберём всё по порядку.

1.1. Использование встроенного режима "Песочница" (Work Profile)
Актуально для Android 8+ с поддержкой Work Profile (управляемый корпоративный профиль).

Что нужно:
  • Android 8.0+
  • Приложение Shelter (open-source, без рекламы)
Шаги:
  1. Установите Shelter из F-Droid (Это магазин open source приложений, достаточно просто в поиске вбить Shelter).
  2. Создайте Work Profile (программа предложит).
  3. Перенесите Max в контейнер через Shelter.
  4. Разрешите только те права, которые нужны мессенджеру.
  5. Все файлы, созданные Max внутри контейнера, будут полностью изолированы.
Плюсы:
  • Open-source.
  • Минимальные системные изменения.
  • Поддержка Android 8+.
Минусы:
  • Нельзя менять IMEI, серийники и ID устройства.
  • Могут быть проблемы с пуш-уведомлениями.

1.2 AppOps + Изоляция разрешений​

Можно использовать AppOps для ограничения прав мессенджера:
  • Блокировка доступа к камере.
  • Ограничение трекинга геолокации.
  • Запрет фоновой работы.
Можно скачать в гугл плей: AppOps

1.3 Изоляция через виртуализацию приложений​


Если хочется полной песочницы, можно использовать виртуальные окружения.

Virtual Android — это бесплатное приложение, которое создаёт полноценную виртуальную машину Android прямо на вашем телефоне.
Оно позволяет запускать мессенджеры, игры и другие приложения в отдельной песочнице, полностью изолированной от основной системы.
Подходит для тестирования, повышения приватности и запуска нескольких копий одного приложения.

Где скачать Virtual Android​

Важно: скачивайте только с официальных источников, чтобы избежать вирусов и модифицированных APK.
Нажмите, чтобы раскрыть...
  • F-Droid (безопасно, open-source):

    F-Droid - Free and Open Source Android App Repository

    F-Droid — это каталог приложений FOSS (Free and Open Source Software — свободное программное обеспечение с открытым исходным кодом) для платформы Android. F-Droid упрощает просмотр, установку и отслеживание обновлений на вашем устройстве.
    f-droid.org

Установка Virtual Android​

Шаг 1. Скачивание и установка
  1. Установите из F-Droid.
  2. Разрешите установку приложений из неизвестных источников:
    Настройки → Безопасность → Установка APK.
  3. Установите Virtual Android.
Шаг 2. Первичная настройка
  1. Запустите Virtual Android.
  2. Приложение создаст виртуальное окружение Android (занимает 1–2 ГБ).
  3. Дождитесь загрузки виртуальной системы.
  4. На экране появится чистый Android, полностью независимый от основной системы.
Шаг 3. Установка мессенджера Max в песочницу
  1. Внутри Virtual Android откройте Google Play (если нужен) или установите APK вручную.
  2. Найдите Max и установите его.
  3. Теперь у вас будет полностью изолированная версия Max.
Virtual Android — это идеальное решение, если вы хотите:
  • Изолировать Max или любой другой мессенджер.
  • Запускать несколько копий приложений.
  • Тестировать программы без доступа к личным данным.
  • Использовать песочницу бесплатно и без root.

ВЫВОД:​

Самое простое Shelter или просто ограничить доступ при помощи AppOps.
Справится в целом любой.)

Для профи, это конечно Virtual Android, но могут-быть сложности с настройкой, долго грузится и т.д.

Изоляция Max на iOS​

С iOS всё сложнее — Apple не даёт нативных песочниц.
Но есть способ добиться изоляции...

Использование встроенного "Screen Time" (Sandbox-like)

Функция Экранное время позволяет ограничить доступ к данным:

Шаги:
  1. Откройте Настройки → Экранное время.
  2. Включите Контент и конфиденциальность.
  3. Ограничьте доступ Max к геолокации, фото, контактам и микрофону.
  4. Ограничьте интернет-доступ через Ограничения контента.
Плюсы:
  • Встроенный метод, не требует взлома.
  • Минимальные риски.
Минусы:
  • Приложение всё равно будет работать в общем профиле.
  • Данные iOS всё равно могут быть доступны.
Удачи!)
